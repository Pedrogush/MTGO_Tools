Deck Workspace

The Deck Workspace is the tabbed area in the centre of the main window. It shows the cards and supporting information for the currently loaded deck.

Tabs

TabDescription
Mainboard Displays the 60-card main deck as a sortable grid. Each row shows the card count, name, mana cost, type, and ownership status (if a collection is loaded). Use the + and − controls in each row to adjust counts when building your own deck.
Sideboard Displays the 15-card sideboard in the same format as the Mainboard tab.
Sideboard Guide Matchup-by-matchup sideboarding notes. See Sideboard Guide.
Deck Notes Free-form text notes saved per deck. See Deck Notes.

Card rows

Hovering over a card row previews the card in the Card Inspector panel. Clicking a row selects it and updates the inspector with full detail including mana cost, type, and card image.

Collection overlay

If you have loaded your MTGO collection (via Settings → Load Collection), each card row is annotated to show how many copies you own. Rows where you own all required copies are highlighted differently from rows where you are missing cards.
