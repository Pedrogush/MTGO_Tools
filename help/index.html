MTGO Tools Help

Welcome to the MTGO Tools help system. MTGO Tools is a Windows desktop application for Magic: The Gathering Online players. It provides metagame research, deck building and editing, opponent tracking, match history analysis, and collection management in one place.

Quick Start

  1. Select a format in the left panel (e.g. Modern, Legacy).
  2. Click an archetype name to load its decklists in the Deck Results panel.
  3. Select a specific deck from the list to view its cards in the Deck Workspace.
  4. Use the toolbar buttons to open the Opponent Tracker, Timer Alert, Match History, or Metagame Analysis windows.
