Settings

Access settings by clicking the ⚙ (gear) button at the top-right of the toolbar. A dropdown menu appears with the following options.

Menu items

ItemDescription
Load Collection Triggers a refresh of your MTGO collection via the MTGOBridge component (.NET). After loading, each card row in the Deck Workspace shows how many copies you own. Requires the .NET bridge to be available.
Download Card Images Opens a dialog to bulk-download card images from Scryfall. You can choose to download images only for cards in your current deck, or for all cards in the database. Downloaded images are cached locally and used by the Card Inspector.
Update Card Database Forces a fresh download of the card metadata database (names, costs, types, oracle text) from MTGJSON. This is required before the Deck Builder can search cards.
Export Diagnostics Exports a diagnostics bundle (logs, settings, version info) to help with bug reports. No personal or account data is included.
Show Tutorial Reopens the first-run tutorial wizard that introduces the main features of MTGO Tools.
Help Opens this help system. You can also press F1 at any time.

Deck data source submenu

OptionDescription
Both Shows decks from both MTGGoldfish and MTGO.com (when MTGO decklists are enabled).
MTGGoldfish Shows only MTGGoldfish decklists. This is the default.
MTGO.com Shows only MTGO.com decklists. Requires the MTGO decklists feature to be enabled via the MTGO_DECKLISTS_ENABLED environment variable.

Language submenu

Choose between English and Português (Brasil). The language applies to all UI labels. A full restart is recommended to apply the change everywhere.

Session state

The following are saved automatically and restored on the next launch:

  • Selected format.
  • Left-panel mode (Research or Builder).
  • Current deck text and zone contents.
  • Window size and position.
  • Language and deck data source selections.
