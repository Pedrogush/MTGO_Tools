Deck Builder

Switch the left panel to Builder mode to search for individual cards and assemble your own deck. Builder mode is accessed by clicking the Builder toggle in the left panel header when a deck is loaded.

Controls

ControlDescription
Search box Type a card name or keyword. Results appear in the list below as you type. The card database must be downloaded before searching (see Settings → Update Card Database).
Result list Click a result to preview it in the Card Inspector.
Add to Main Adds one copy of the selected card to the Mainboard.
Add to Side Adds one copy of the selected card to the Sideboard.
Mana Keyboard Opens the floating mana-symbol keyboard. Click a symbol to insert it into the search field, which allows searching by mana cost.
Back to Research Returns the left panel to Metagame Research mode.

Editing counts

Once a card is in the deck, use the + / − buttons in the Deck Workspace card rows to adjust counts. Right-clicking a row provides additional options such as removing the card entirely.

Radar analysis

Click Radar (available in Builder mode) to open the radar dialog, which shows how frequently cards appear across archetypes in the current format. This helps identify format-staple cards when building.
