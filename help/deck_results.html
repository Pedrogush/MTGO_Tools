Deck Results & Actions

The Deck Results panel is at the bottom-right of the main window. After you click an archetype in the Metagame Research panel, this area lists the individual decklists available for that archetype.

Summary text

A brief summary line above the list shows the archetype name, win rate, and meta share where available.

Deck list

Each item in the list represents a specific tournament deck for the selected archetype. Click a deck to load it into the Deck Workspace.

Action buttons

ButtonDescription
Today’s Average Computes a card-by-card average across all available decklists for the current archetype and loads the resulting averaged deck into the workspace. Useful for seeing the typical build of an archetype.
Copy Copies the current deck list to the clipboard in MTGO-compatible plain-text format (e.g. 4 Lightning Bolt). You can paste this directly into MTGO.
Load Deck Opens a file browser to load a deck from a local .dek or plain-text file.
Save Deck Saves the current deck to a file and optionally to the local MongoDB database (if configured). The saved deck persists across sessions.
