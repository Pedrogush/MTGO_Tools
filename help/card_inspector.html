Card Inspector

The Card Inspector sits on the right side of the main window. It shows detailed information about the currently focused card.

Displayed information

  • Card image — downloaded from Scryfall. Images are cached locally after the first download. If an image has not been downloaded yet, a placeholder is shown and the download starts automatically.
  • Card name and mana cost with mana symbols.
  • Type line (creature type, instant, sorcery, etc.).
  • Oracle text — the card’s rules text.
  • Power / toughness or loyalty where applicable.
  • Printing selector — a dropdown to choose between different printings of the same card. Changing the printing updates the displayed image.

Triggering the inspector

The inspector updates whenever you:

  • Hover over a card row in the Deck Workspace.
  • Click a card row to select it.
  • Click a search result in the Deck Builder.

Image downloads

Card images are downloaded on demand and stored in the local image cache. You can also pre-download images in bulk via Settings → Download Card Images.
