Metagame Research Panel

The Metagame Research panel occupies the left side of the main window. It lets you browse competitive archetypes for any supported MTGO format, using data fetched from MTGGoldfish.

Controls

ControlDescription
Format dropdown Selects the competitive format (Modern, Legacy, Pioneer, Standard, Pauper, Vintage, etc.). Changing the format clears the current archetype list and fetches new data.
Search box Filters the archetype list in real time. Type part of an archetype name to narrow the list.
Archetype list Displays the archetypes available for the selected format. Click an entry to load its decklists into the Deck Results panel.
Reload Archetypes Forces a fresh fetch from MTGGoldfish, bypassing the local cache.

Data source

Archetype data is sourced from MTGGoldfish and cached locally. You can control whether the app also shows decks from MTGO.com via Settings → Deck data source. By default only MTGGoldfish data is shown.
