Match History

The Match History window parses your MTGO GameLog files and displays a summary of recent match results. Click Match History in the toolbar to open it.

GameLog location

MTGO writes GameLog files to your Windows user profile folder, typically at:

%AppData%\Local\Apps\2.0\...\Magic Online\GameLogs\

The application searches the default location automatically. If your logs are in a different location, you can browse for the folder using the folder picker in the Match History window.

Displayed information

  • Date and time of each match.
  • Opponent username.
  • Match result (Win / Loss).
  • Game-by-game breakdown where available.
  • Format and event type parsed from the log.

Filtering

Use the search or filter controls at the top of the window to narrow the history by date range, opponent name, or result.
