Timer Alert

The Timer Alert window provides a configurable countdown timer. Use it to alert yourself before time runs out in a MTGO round or challenge.

Opening the timer

Click Timer Alert in the toolbar. The window floats above other windows so it remains visible while you play.

Controls

ControlDescription
Duration field Set the countdown duration in minutes. Common values: 50 minutes for a MTGO match, 25 minutes for a half-round warning.
Start / Stop Starts or stops the countdown. The display updates every second.
Reset Resets the countdown to the configured duration without starting it.

Alert behaviour

When the countdown reaches zero, the timer window flashes and plays a system alert sound to attract your attention.
